**プロジェクト名称**
ポータブル会計DB

**プロジェクト概要**
Google スプレッドシート等で管理されている会計データを基に、ブラウザ単体で財務状況を可視化するツールです。 貸借対照表（B/S）や損益計算書（P/L）の自動生成に加え、総勘定元帳の閲覧や固定資産の減価償却シミュレーションなど、高度な会計分析機能をポータブルに提供することを目的としています。

**主要な技術スタック**
言語: HTML5, CSS3, JavaScript (Vanilla JS)
フレームワーク・ライブラリ: 使用なし（外部依存のないスタンドアロン構成）
UI/UX: CSS Grid, Flexbox, CSS Variables
データ形式: 内部定数に埋め込まれた JSON 形式の仕訳データ

**ファイル一覧と解説**
主要なファイルとその責務について解説します。

index.html
役割: アプリケーションのフロントエンド、ビジネスロジック、およびデータストアの統合。
責務:
集計エンジン: rawData（仕訳帳）を走査し、現預金残高や売上・経費を動的に算出。
財務諸表出力: 複式簿記の原則に基づき、B/S と P/L をリアルタイムにレンダリング。
資産管理: 耐用年数に応じた定額法による減価償却計算と、決算整理仕訳の自動提案。
データポータビリティ: 次年度期首残高の CSV エクスポート機能。

**エントリーポイント**
プログラムは index.html をブラウザで開くことで開始されます。

動作の流れ:
初期化: window.onload = init; により、スクリプト内の init() 関数が発火します。
データ整合性チェック: 借方と貸方の合計が一致しているか（貸借平均の原理）を確認します。
財務分析の実行: 埋め込まれた仕訳データを解析し、経営ダッシュボードのグラフ、総勘定元帳、財務諸表をそれぞれ生成します。

UI反映: ユーザーのタブ切り替え操作（「経営ダッシュボード」「総勘定元帳」等）に応じて、計算済みの各ビューが表示されます。
